# Numeric comparisons

```
5<10
5<=10
5>10
5>=9
5==9
```

# string comparisons

```
"a"<"b"
"d">"c"
"cat=="cat"
```

# if

```
if 10 < 20:
    print("true")

if not "dog" == "cat":
    print( "They are not the same")
```

# if then else

```
if 23 % 5 == 0:
    print( "divisible by 5")
else:
    print( "not divisible")
```
